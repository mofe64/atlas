# Atlas UI

Atlas UI is the operator-facing React application.

This skeleton uses:

- React
- TypeScript
- Vite
- Tailwind CSS

Run locally:

```sh
nvm use
npm install
npm run dev
```

Build:

```sh
nvm use
npm run build
```

The Vite dev server proxies `/api` to the local backend at:

```text
http://127.0.0.1:8080
```
