# Archived stack scripts

These scripts are retained only as a historical reference for installations
that predate the local-first Atlas Native and packaged Atlas Agent topology.
They are unsupported, are not part of any current install or development
workflow, and may refer to source trees, services, protocols, or infrastructure
that no longer exist in this repository.

- `root/` contains the former ground-computer orchestration and protocol
  generation scripts.
- `onboard/` contains the former Raspberry Pi install, launch, status, video,
  routing, and cleanup scripts.

Do not use these scripts for a new install or upgrade. Follow
[`atlas-agent/INSTALLATION.md`](../../../atlas-agent/INSTALLATION.md) for the
supported onboard deployment runbooks.
